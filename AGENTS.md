## Package Manager

This repository uses `pnpm` exclusively.
When installing dependencies or running package scripts, prefer `pnpm` over `npm` or `yarn`.

Examples:

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm db:migrate`
